## 直觉说明（一个一分钟理解的例子）
在进入框架定义之前，先用一个简单的例子说明“慈悲的测量”在现实中的含义。

想像你一个人走在夜路上。你手里的手电筒，就是“测量”。
它不是照亮整个世界，它只照亮你下一步能看见的那一小块路。

风声、黑暗、狗叫、各种不确定的动静，都是噪音。
情绪如恐惧本身不是噪音，恐惧会把噪音放大。

当恐惧变强，你会更急着决定，手电筒照的范围会变小，你能看见的路会变少，你能犹豫和判断的时间也会被压缩。
最后你走的那条路，不是最好的路，而是最先被照到、最容易下脚的那条路。

这就是“测量”在现实中的样子——它决定的不是世界长什么样，而是你能看见哪些选择。

而“慈悲”，
不是善良。

慈悲，是刻意放慢这道手电筒，在噪音和情绪逼迫你赶快决定之前，为自己保留一点时间，让更多可能的路，来得及被照见。
这就是我说的——慈悲的测量。
